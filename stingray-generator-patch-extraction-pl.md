# stingray-generator-patch-extraction-plan.md

## Recommended patch order

1. Move no-schema option/section/display-order fixes into `stingray_master`, `section_master`, `option_variant_status`, `rule_mapping`, and `price_rules`.
2. Add minimal support sheets: `option_aliases`, `choice_groups`, `choice_group_members`, `option_display_overrides`.
3. Add minimal columns: `rule_mapping.active/action/disabled_reason`, `lt_interiors.active_for_stingray/requires_r6x/included_option_id`, `PriceRef.component_type/component_label`.
4. Re-run generator before removing Python constants; generated app data should diff to zero except intentional workbook metadata changes.
5. Do not touch Grand Sport behavior until the Stingray extraction is stable.
